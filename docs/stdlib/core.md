# Core Functions

Core runtime functions available in Atlas.

## print

```atlas
fn print(value: any) -> Null
```

Prints a value to stdout. Returns Null.

**Parameters:**
- `value` - Any value to print

**Returns:** `Null`

## len

```atlas
fn len(value: []T) -> number
```

Returns the length of an array, string, or collection.

**Parameters:**
- `value` - Array, string, or collection

**Returns:** `number` - Length of the value

## str

```atlas
fn str(value: any) -> string
```

Converts a value to its string representation.

**Parameters:**
- `value` - Any value

**Returns:** `string` - String representation
