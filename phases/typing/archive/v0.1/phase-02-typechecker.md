# Phase 02 - Type Checker

## Objective
Enforce strict typing rules on the bound AST.

## Inputs
- `Atlas-SPEC.md`
- `docs/diagnostics.md`

## Deliverables
- Type inference for locals.
- Explicit param/return type enforcement.
- Operator typing rules.

## Steps
- Implement type checking for expressions and statements.
- Enforce explicit types for function params/returns.
- Map type errors to diagnostic codes.

## Exit Criteria
- All type error codes exercised in tests.
