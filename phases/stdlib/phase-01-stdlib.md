# Phase 01 - Standard Library

## Objective
Implement the v0.1 standard library.

## Inputs
- `docs/stdlib.md`

## Deliverables
- `print`, `len`, `str` with correct types.
- Error handling for invalid inputs.

## Steps
- Implement stdlib functions in runtime.
- Add tests for normal and error cases.

## Exit Criteria
- Stdlib functions work in both interpreter and VM.
