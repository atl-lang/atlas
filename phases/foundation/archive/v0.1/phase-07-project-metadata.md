# Phase 07 - Project Metadata

## Objective
Establish baseline repository metadata and legal/behavioral policies.

## Inputs
- `docs/guides/code-quality-standards.md`
- `docs/guides/code-quality-standards.md`

## Deliverables
- `LICENSE` (default: MIT unless decided otherwise).
- `CODE_OF_CONDUCT.md` (lightweight, standard template).
- `CONTRIBUTING.md` references engineering/style docs.

## Steps
- Choose license and document rationale.
- Add code of conduct template.
- Update CONTRIBUTING to include test and formatting steps.

## Exit Criteria
- Metadata files exist and are consistent with project standards.
- README links to these files.
