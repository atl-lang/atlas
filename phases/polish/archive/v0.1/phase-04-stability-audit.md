# Phase 04 - Stability Audit

## Objective
Run a full audit of test stability and diagnostic consistency.

## Inputs
- Test suite results

## Deliverables
- Report on flaky tests (should be none).
- Diagnostics format check.

## Steps
- Run full test suite multiple times.
- Confirm diagnostic format stability.

## Exit Criteria
- All tests deterministic, no known flakiness.
