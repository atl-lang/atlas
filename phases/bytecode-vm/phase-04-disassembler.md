# Phase 04 - Bytecode Disassembler

## Objective
Implement a bytecode disassembler for debugging and tests.

## Inputs
- Bytecode format definitions

## Deliverables
- Disassembler that prints readable instruction listings.
- CLI option to dump bytecode (`atlas build --disasm`).

## Steps
- Implement disassembler formatting.
- Add tests for stable output.

## Exit Criteria
- Disassembler output stable and testable.
