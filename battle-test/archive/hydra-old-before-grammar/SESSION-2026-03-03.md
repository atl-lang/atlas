## Key Learnings

### Match Expressions
Most common error - can't use code blocks:
```atlas
// ✅ Correct
let value: string = match result {
    Ok(data) => data,
    Err(_) => "default"
};

// ❌ Wrong
match result {
    Ok(data) => { return data; }  // ERROR
}
```

### File API
Throws on error, no Results:
```atlas
writeFile(path, content);  // throws, not Result
```

### exec() Behavior
Synchronous, blocks until complete:
```atlas
exec(["sleep", "2"]);  // Blocks for 2 seconds
```
