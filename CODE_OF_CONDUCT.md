# Code of Conduct

## Our Pledge

We are committed to providing a welcoming and inclusive environment for all contributors.

## Our Standards

Expected behavior:
- Be respectful and professional
- Accept constructive feedback gracefully
- Focus on what is best for the project
- Show empathy towards other community members

Unacceptable behavior will not be tolerated.

## Enforcement

Project maintainers are responsible for clarifying standards and will take appropriate action in response to unacceptable behavior.

## Scope

This Code of Conduct applies within project spaces and when representing the project in public spaces.

## Contact

Report issues to project maintainers via GitHub issues.

## Attribution

This Code of Conduct is adapted from standard open source community guidelines.
